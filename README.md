# Cadastro de Usuários (React & Node.js) 

## Descrição  
Aplicação full-stack desenvolvida para gerenciar cadastro de usuários, permitindo criar, visualizar e deletar registros. O front-end, construído em React, oferece uma interface dinâmica e fluida, enquanto o back-end em Node.js gerencia a lógica e persistência dos dados. Avatares aleatórios são exibidos para cada usuário, integrando uma API externa para enriquecer a experiência.

## Tecnologias Utilizadas  

- **Front-end:** React.js, CSS e HTML  

- **Back-end:** Node.js, Express.js (presumido)  

- **APIs:** Serviço de avatares aleatórios para fotos de usuários  

- JavaScript para lógica e manipulação de dados

## Funcionalidades  

- Cadastro de novos usuários com dados essenciais  

- Listagem dinâmica de usuários cadastrados com avatares aleatórios  

- Exclusão de usuários com atualização instantânea da interface  

- Interface responsiva e focada em usabilidade para melhor experiência do usuário
